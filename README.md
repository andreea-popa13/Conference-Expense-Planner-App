# Conference Event Planner Application

An interactive web application built with **React** and **Vite**, developed as part of the IBM Certification program. This app allows users to efficiently plan and manage budgets for events and conferences.


##  Key Features
* **Venue Selection:** Choose locations and manage venue-related costs.
* **Catering Management:** Add and customize meal services for attendees.
* **Add-on Support:** Select additional equipment like projectors and microphones.
* **Real-time Calculation:** Automatically calculates the total expenses as you add or remove items.

##  Tech Stack
* **React.js** - Frontend library for the user interface.
* **Vite** - High-performance build tool.
* **JavaScript (ES6+)** - Application logic.
* **CSS3** - Custom styling and layout.
